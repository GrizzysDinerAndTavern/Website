# Grizzy's Diner & Tavern Website

Static one-page website for Grizzy's Diner & Tavern.

## Required structure

- `index.html` must remain in the repository root.
- Keep the `assets` folder beside `index.html`.
- Netlify build command: leave blank.
- Netlify publish directory: `.`

Do not deploy the ZIP file itself as the site root. Extract it first or push these files to GitHub.
